# TLC Trial Team — Website Redesign

A modern static-HTML mockup of [tlctrialteam.com](https://tlctrialteam.com/),
designed and built by [Ensotography](https://ensotography.com).

**Stack**
- Plain HTML / CSS / JS
- Cormorant Garamond + Inter (Google Fonts)
- Lenis smooth scroll (CDN)
- Static hosting on Vercel (preview URL auto-deploys on push)

**Live preview**
- Production: TBD (set after first Vercel deploy)
- Per-PR preview: auto-generated by Vercel

## Local preview

```bash
# Just open in Chrome
open index.html
```

## Project structure

```
.
├── index.html               # The homepage
├── PITCH.md                 # Sales talking points (for Tania pitch)
├── vercel.json              # Static-site config
├── push-tlc.command         # Double-click to push to GitHub
└── assets/
    ├── css/styles.css       # Design system + page styles
    ├── js/main.js           # Lenis smooth scroll + scroll reveals
    └── img/                 # Logo, attorney portrait, badges
```

## Deploying changes

1. Edit any file (the design system lives in `assets/css/styles.css`).
2. Open `index.html` in Chrome to verify locally.
3. Double-click `push-tlc.command` to push to GitHub.
4. Vercel auto-deploys; the preview URL updates within ~30 seconds.
