## 4. Brand decisions (locked)

Don't relitigate without an explicit ask.

### Palette
- `--white` — `#FFFFFF` (primary background)
- `--paper` — `#FAF6EE` (warm cream — soft section backgrounds)
- `--paper-2` — `#F2EBD9` (deeper cream)
- `--ink` — `#1A1A1A` (near-black text)
- `--ink-soft` — `#3F3F3F` (secondary text)
- `--mute` — `#8B847A` (tertiary text)
- `--teal` — `#478F8E` (TLC's actual brand teal — NOT a deeper one)
- `--teal-deep` — `#2F6968` (hover / deep accent)
- `--gold` — `#C9A84F` (refined cream-gold accent)

The `#478F8E` teal is pulled from TLC's actual logo + current CSS. The
warmer `#FAF6EE` paper matches their existing site's section backgrounds.

### Typography
- Display: Cormorant Garamond, weights 300/400/500, italics 300/400.
  Use weight 300 for hero (mega-size), 400 for h1/h2, 500 for h3+.
  Italics carry emotional emphasis — wrap key words in `<em>` and
  color with `--teal`.
- Body: Inter, weights 300/400/500. Body copy is 400 at 1.025rem.
  Use weight 300 for `.lede` paragraphs (more breathing).
- Eyebrows: Inter 500, 0.7rem, letter-spacing 0.22em, uppercase, teal.
  No mono fonts — too engineering-y for this brand.

### Layout principles
- White-dominant. Section padding `clamp(6rem, 12vw, 12rem)` — generous.
- Asymmetric grids (`5fr 6fr` not `1fr 1fr`).
- NO boxy grids of identical cards. Practice areas are a numbered
  vertical list. Areas served are a typographic flowing line. Case
  results are ONE giant `$1.4M` next to a thin supporting list.
- Cards (when used) have no visible borders — separation comes from
  background contrast (paper vs white) and whitespace.
- Buttons: pill-shaped (radius 999px), filled black-on-white as primary,
  no gradients.

### Motion
- Lenis smooth scroll on by default.
- Scroll reveals: opacity 0 → 1, translateY 28px → 0, 1100ms ease.
- Stagger: 40ms delay per child (max 9 children — `.reveal-stagger`).
- No fade-based word rotators. Masked vertical roll only if rotator is
  re-introduced.

## 6. Deployment workflow

1. Edits happen in the sandbox (Cowork) on the workspace folder
   `~/Documents/Claude/Projects/ENSOTOGRAPHY AI/tlc-trial-team-redesign`.
2. Sandbox cannot push to GitHub (no credentials).
3. Giovani double-clicks `push-tlc.command` in Finder.
4. A native macOS dialog asks for the commit message.
5. The wrapper sources `scripts/push.sh`, which:
   - `git add -A`
   - `git commit -m "$MSG"`
   - `git push origin main`
   - Echoes the GitHub + Vercel URLs
   - Fires a native macOS notification
6. Vercel auto-deploys from `main` in ~30 seconds.

**The Cowork mount strips execute bits.** That's why `push-tlc.command`
must NOT be edited — recreating it removes its `+x` bit and breaks
double-click. The actual push logic in `scripts/push.sh` is invoked via
`bash`, so it doesn't need `+x`.
